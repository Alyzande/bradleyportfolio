MERN Full Stack Recruitment App

The final project for 'The Developer Academy' web development bootcamp was created following agile project management methods with Jira and Slack for communication. Our team used technologies HTML CSS Bootstrap Node Mongo Database React Express JavaScript and Heroku. Our full stack application was created to showcase graduates of the client's bootcamps, built on a MongoDB backend. Graduates create, and update their profiles which are available for employers to browse and select according to their choice of filters.

Shoutcast Player

LiteRECORDS started out as a pet project in 2010 become my classroom and foundation for learning large-scale web development. It was here I began my understanding of JavaScript, PHP and XML with vBulletin hack. This project displays the Shoutcast server details from XML, fetches the user details from MySQL database and refreshes the details plus online indication via AJAX.

Weather API

An ambitious project using HTML CSS Bootstrap and React to connect OpenWeather API. I learned about React, and API handling. Working as a team using 'ping pong' code technique we created a single page app with a 7 day weather forecast and hourly forecast. We fulfilled the project brief and exceeded expectations by adding wind speed & direction with subtle icons, sunrise, sunset, and location interactivity. I would like to add geolocation handling.

OOP Text Adventure game

Using HTML CSS and JavaScript, I built this mobile-first OOP text game on one page, with functionality to explore rooms, move between rooms, interact with characters, with win/lose conditions. I enjoyed learning object orientated programming in JavaScript. I improved the project by adding a command instructions, objects and cue icons to build a more user friendly interface.

JavaScript Quiz

A single page game built in JavaScript to show DOM node hide/show capabilities together with variables that count score for a quiz on music. I completed the project and also added some extra functionality to enhance user interaction.
